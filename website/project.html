Smart

The core of the project runs on the Raspberry Pi 2 B+ and it's able to take decisions, interpretate messages and add new configurations.

Actuator

The decisions taken by the Raspberry are actuaded by the Arduino Uno that waits for commands and replies.

Communications

The communications between the elements inside the network is made through mqtt.
